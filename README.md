# 基于Dify的NLP2SQL智能数据查询助手

一个基于Dify低代码平台搭建的自然语言转SQL数据查询工具，支持用户用日常提问自动生成并执行MySQL查询，降低非技术人员的数据使用门槛。（数据仅模拟测试）

## 功能特性
- 自然语言提问自动生成可执行SQL语句
- 支持平均分、及格率、成绩分布等多维度数据统计
- 完整工作流：用户输入 → SQL生成 → 数据库执行 → 结果整理
- 低代码配置，开箱即用

## 技术栈
- 低代码平台：Dify
- 数据库：MySQL 8.0
- 容器部署：Docker
- 大模型：通义千问Qwen
- 工具链：Git、SQL

## 快速部署
1.  安装Docker并部署Dify平台
2.  运行 `init\\\_database.sql` 创建数据库与测试数据
3.  在Dify中导入 `07-nlp2sql 数据库查询助手.yml` 工作流
4.  配置本地MySQL数据库连接凭据
5.  发布应用，即可开始使用
